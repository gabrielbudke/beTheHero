# Be The Hero
> Projeto resultado da Semana OmniStack #11

## Inicialização

* Iniciando o servidor
```
cd backend
npm install || yarn
npm start || yarn start 
```

* Iniciando a página web
```
cd frontend
npm install || yarn
npm start || yarn start 
```

* Iniciando o app mobile
```
cd mobile
npm install || yarn 
```
